This WordPress plugin extends the WooCommerce plugin with the Swedish translation.

== Description ==

Pluginet aktiverar svenska språkfiler från Transifex för WooCoomerce. Detta är ett tillägg och kräver att WooCommerce är installerat.

== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your 
WordPress installation and then activate the Plugin from Plugins page.
